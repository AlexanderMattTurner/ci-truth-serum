- New Tier 2 hook `check-path-gate-deps`: verifies every decide-gated job's static dependencies (local composite actions, `.github/scripts/` helpers one `source` hop deep, and `# gate-deps:`-declared paths) are covered by the decide job's path filters, so a PR changing only a dependency can't silently skip the job and green-light its `always()` reporter. Suppress a single dependency with `# path-gate-ok: <dep> <reason>`; a `uses: ./<dir>` naming a missing local action is its own hard error. Registered `always_run` so dependency-side changes (a new file under a covered dir) re-fire the check.
