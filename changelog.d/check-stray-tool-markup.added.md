- `check-stray-tool-markup` (Extras): bans stray agent tool-call markup accidentally committed into a file — a line that is entirely a leaked closing/opening tool-call tag (`</invoke>`, an opening `invoke`/`parameter` tag, `<function_calls>`, a bare closing `content` tag, and the `antml:`-prefixed variants). Only whole-line bare tags are flagged; inline mentions, inline-code spans, and fenced code blocks are skipped. Suppress a genuine case with `allow-stray-markup: <reason>` on the line above.
