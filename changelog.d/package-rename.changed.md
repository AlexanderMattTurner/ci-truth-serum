- **Breaking:** the installed Python package is renamed from `hooks` to `ci_truth_serum`, so a consumer repo with its own top-level `hooks/` directory can no longer shadow it (which made every hook `ImportError`). All `entry:` lines in `.pre-commit-hooks.yaml` change from `python -m hooks.<module>` to `python -m ci_truth_serum.<module>`, and the `sync-required-checks` / `release-canary` console scripts move to the `ci_truth_serum.` module path. Consumers pinning a specific `rev` and running the hooks by module path are unaffected (pre-commit invokes the packaged entries); anyone invoking `python -m hooks.<check>` directly must switch to `python -m ci_truth_serum.<check>`.
