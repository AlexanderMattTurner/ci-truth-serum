- `check-env-symmetry` (Extras): scans the whole tracked tree for every env var matching a `--prefix` and flags any that is written but never read (or read but never written) — the half-finished-rename signature where the reader silently sees an unset value. Conservative extraction keeps false positives near zero; dynamically-composed names are skipped, and an out-of-band var opts out with `# env-symmetry-ok: <VARNAME> <reason>`. Needs `args: [--prefix, <PREFIX>]`; not part of a tier aggregate.
